Steps to install 
1 set up databease 
2 install pip using sudo apt-get install python-pip
3 install virtual environment  using python3 -m pip install --user virtualenv
4 create  virtual env  with python3 -m virtualenv env
5 activate virtual environment with source env/bin/activate
6 install requirements using pip install -r requirements.txt
for mysql client instal sudo apt-get install python3-dev default-libmysqlclient-dev build-essential then pip install mysqlclient
